We have implemented, five models using both python and julia language, and used jupyter notebook and pluto environment:

- ANN model : 2, 3 and 5 layers ( julia - pluto)
- SVM model (python - jupyter notebook)
- LOG model (python - jupyter notebook)
- Decision Tree model (python - jupyter notebook)
- KNN model (python - jupyter notebook)

To run these models, you need, firstly to install, all dependencies.

* Julia dependencies
--------------------
- DelimitedFiles
- Plots
- Statistics
- LinearAlgebra
- Flux
- PlutoUI


* Python dependencies
---------------------
- pandas
- numpy
- seaborn
- matplotlib
- sklearn

After installing, all dependencies, you can run each source file, according for their environment respectively.
